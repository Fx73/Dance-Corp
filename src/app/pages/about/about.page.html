üìÅ File Reader

Reader accepts stepmania files, SM and SSC.

It create an Essc file (extended scc) which is actually an scc with unused field remove and one change : all special notes are converted to integer as following

0 = no note

1 = tap note

2 = hold head

3 = Hold/roll/Minefield end

4 = Roll head

5 = Mine (was M)

6 = Minefield head(was N)

7 = Fake(was F)

8 = Hidden(was H)

Score calculation

The score is calculated based on a perfect score of 1 000 000. It is divided between tap and hold based on hold duration (100% time hold would mean 50% score in hold). Then each note is a subdivision of the score and each hold is following the hold time to get its score
